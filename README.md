# Formulário de Matrícula - Estrelas do Amanhã

Projeto desenvolvido durante os estudos de **HTML5** e **CSS3** com o objetivo de praticar a criação de formulários completos, organização modular dos estilos e construção de interfaces modernas e responsivas.

## Acesse o projeto

 **https://mariasmorais.github.io/Formulario-de-matricula/**

---

## Sobre o projeto

O projeto consiste em uma página de matrícula para uma escola fictícia de educação infantil chamada **Estrelas do Amanhã**.

O formulário reúne as principais informações necessárias para realizar a matrícula de uma criança, incluindo:

- Informações pessoais;
- Endereço residencial;
- Dados do responsável;
- Upload da certidão de nascimento;
- Seleção do turno de estudo;
- Escolha de atividade esportiva;
- Aceite dos termos e condições.

O desenvolvimento teve como foco a organização do código, HTML semântico, estilização moderna e boas práticas na construção de formulários.

---

## Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Variáveis CSS
- Pseudo-classes (`:hover`, `:focus`, `:checked` e `:has`)
- Organização modular dos arquivos CSS

---

## Funcionalidades

- Formulário dividido em seções utilizando `<fieldset>` e `<legend>`;
- Campos de texto, data, seleção e área de texto;
- Upload personalizado de arquivos;
- Botões de rádio personalizados;
- Checkbox personalizado;
- Feedback visual para foco, seleção e validação dos campos;
- Layout dividido entre formulário e painel ilustrativo.
